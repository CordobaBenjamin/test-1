How to install

1) Follow the step to install react native and node.js on your System

https://facebook.github.io/react-native/docs/getting-started.html

After react native installation completed

>> Open Terminal in your project folder
>> Hit npm install command (this command will install all node_modules in your project folder without node_modules this project will not run)

How to run project?

>> react-native run-android

>> react-native run-ios

Code structure

Root folder has App.js file. which has all screen nevigation code.

After that all screen code located at src directory. it has main 3 folder component which has like buton. than helper which has all icon and text configuration. and last screens has all screen of app.

How to rename project?

react-native-rename - Rename react-native app with just one command
